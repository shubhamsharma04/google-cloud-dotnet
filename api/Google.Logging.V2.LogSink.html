Filter

An advanced logs filter that defines the log entries to be exported. The filter must be consistent with the log entry format designed by the outputVersionFormat parameter, regardless of the format of the log entry that was originally written to Cloud Logging. Example: "logName:syslog AND severity>=ERROR".

Name

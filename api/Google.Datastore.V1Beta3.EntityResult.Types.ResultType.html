Enum EntityResult.Types.ResultType

Specifies what data the 'entity' field contains. A ResultType is either implied (for example, in LookupResponse.found from datastore.proto, it is always FULL) or specified by context (for example, in message QueryResultBatch, field entity_result_type specifies a ResultType for all the values in field entity_results).

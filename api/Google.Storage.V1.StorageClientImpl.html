Class StorageClientImpl

Wrapper around StorageService to provide simpler operations.

Inheritance
System.Object
StorageClient
StorageClientImpl
Namespace:Google.Storage.V1
Assembly:Google.Storage.V1.dll
Syntax
public sealed class StorageClientImpl : StorageClient
Remarks

This is the "default" implementation of StorageClient. Most client code should refer to StorageClient, creating instances with

Create(GoogleCredential) and

CreateAsync(GoogleCredential). The constructor of this class is public for the sake of constructor-based dependency injection.

ApplicationName

The default application name used when creating a StorageService. Defaults to "gcloud-dotnet"; must not be null.

Remarks

Most applications will never want to set this, which is why it's in this class rather than

StorageClient.

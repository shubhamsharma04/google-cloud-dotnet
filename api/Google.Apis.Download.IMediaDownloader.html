DownloadAsync(String, Stream, CancellationToken)

Downloads asynchronously the given URL to the given stream. This download method supports a cancellation token to cancel a request before it was completed.

Remarks

In case the download fails Exception will contain the exception that cause the failure. The only exception which will be thrown is

System.Threading.Tasks.TaskCanceledException which indicates that the task was canceled.

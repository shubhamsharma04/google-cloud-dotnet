Fields

Name Description
EndOfAudio

The end of the audio stream has been reached. and it is being processed.
EndOfSpeech

Speech has ceased to be detected in the audio stream.
EndOfUtterance

This event is only sent when single_utterance is true. It indicates that the server has detected the end of the user's speech utterance and expects no additional speech. Therefore, the server will not process additional audio. The client should stop sending additional audio data.

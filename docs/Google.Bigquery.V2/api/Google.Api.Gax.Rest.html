IPagedAsyncEnumerable<TResponse, TResource>

An asynchronous sequence of resources obtained via API responses. Application code can treat this as a simple sequence (with API calls automatically being made lazily as more results are required), or call AsPages() to retrieve a page at a time, potentially with additional information.

IPagedEnumerable<TResponse, TResource>

A sequence of resources obtained via API responses, each of which contributes a page of resources. Application code can treat this as a simple sequence (with API calls automatically being made lazily as more results are required), or call AsPages() to retrieve a page at a time, potentially with additional information.
